# 📋 ToDo App

رابط المشروع: https://yomnaelsafty.github.io/Todo-app/

تطبيق ToDo بسيط ومميز لإدارة المهام اليومية.

---

## 🚀 الوصف

تطبيق **ToDo App** هو واجهة ويب بسيطة تساعدك على:
- إضافة مهام جديدة 📝
- وضع علامة على المهام المكتملة ✔️
- حذف المهام غير المرغوب فيها ❌

التطبيق مبني باستخدام **HTML، CSS، JavaScript** فقط، بدون Back‑end أو قاعدة بيانات — يتم حفظ المهام في الـ LocalStorage 👌.

---

## 🛠️ المميزات

✔️ إضافة مهام جديدة  
✔️ الاحتفاظ بالمهام بين الجلسات باستخدام LocalStorage  
✔️ إمكانية تمييز المهام كمكتملة  
✔️ حذف كل مهمة على حدة  
✔️ تصميم نظيف وسهل الاستخدام

---

## 🧠 كيف يعمل

1. اكتب اسم المهمة في الحقل.
2. اضغط “Add” لإضافتها لقائمة المهام.
3. اضغط علامة ✔️ لوضع المهمة كمكتملة.
4. اضغط علامة ❌ لحذف المهمة.

---
